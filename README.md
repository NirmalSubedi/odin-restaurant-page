# Restaurant 
Restaurant page to practice DOM manipulation by dynamically rendering it.

[Visit Website!](https://nirmalsubedi.github.io/odin-restaurant-page/)

## Which skills were demonstrated?
- Using EcmaScript 6 modules with imports and exports.
- Installing npm packages and configuring `package.json` to track dependency modules.
- Bundling with webpack for it to handle dependency graph and entry point.
- Configuring webpack with `webpack.config.js` file to use loaders( css and html ), plugins( html ), and devServer.
